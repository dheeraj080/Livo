# 📝 TaskMaster Pro Frontend

A high-performance, premium React client for the TaskMaster Pro API.

## 🚀 Getting Started

### 1. Installation
```bash
npm install
```

### 2. Configuration
The frontend expects the backend to be running at `http://localhost:8080`.
If your backend is on a different port, update `src/api/axios.ts`.

### 3. Development
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view your app.

## 🛠 Tech Stack
- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **HTTP Client**: Axios
- **Form Handling**: React Hook Form + Zod
